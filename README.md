# Gesture-controlled-image-Filter

Application made using OpenCV and Mediapipe Hand Landmarks and Euclidean distance calculations to recognise hand gestures in real time and apply different image processing effects using OpenCV.

*Features* 

1. pinch gesture - Gaussian Blur
2. Open Palm - Grayscale
3. Luck gesture - Pixelation

Technologies :
1. Python
2. OpenCV
3. Mediapipe
4. Numpy
